# 🚀 PyTorch Fundamentals & Deep Learning Practice

This repository documents my journey of learning **PyTorch** and building a strong foundation in **Deep Learning concepts** from scratch.

Instead of just using high-level APIs, I focus on understanding **how things work internally** — especially autograd, loss functions, and tensor operations.

---

## 📌 What this repo contains

This is not a “project-based” repository.

It is a **concept-based learning repo** where I:

* Experiment with core PyTorch functionalities
* Break down important deep learning concepts
* Write code + explanations for better understanding

---

## 🧠 Topics Covered

### 🔹 1. Tensors & Operations

* Creating tensors
* Basic operations
* Broadcasting
* Gradient tracking (`requires_grad`)

---

### 🔹 2. Autograd (Core Focus)

* Computation graphs
* How `.backward()` works
* Scalar vs vector outputs
* Manual gradient passing
* Gradient flow visualization

---

### 🔹 3. Loss Functions

* Binary Cross Entropy (BCE) from scratch
* Why we use `log` in BCE
* Numerical stability issues
* Why we use `clamp()`

---

### 🔹 4. Gradient Behavior

* Effect of `.sum()` vs `.mean()`
* How batch size affects gradients
* Stability considerations
